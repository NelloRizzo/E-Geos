Path di invocazione

  • Pattern per le invocazioni REST: /api/v1/{controller}/{params}
    Esempio:
    GET /api/v1/populate/cities => carica sul database l'elenco delle città.
    GET /api/v1/cities => ottiene l'elenco delle città in archivio.
    GET /api/v1/cities/1 => ottiene i dati della città con id = 1 .
    GET /api/v1/cities/1/geo => ottiene i dati della città con id = 1 e la posizione geografica prelevata da Google Maps (se disponibile).
